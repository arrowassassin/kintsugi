# kintsugi-intercept

Interception adapter library for [Kintsugi](https://github.com/arrowassassin/kintsugi): the `$PATH` shim, the Claude Code hook bridge, and the `kintsugi-exec` MCP server. All normalize to one `ProposedCommand`.

This is an internal crate. Install the [`kintsugi`](https://crates.io/crates/kintsugi) crate to get all binaries.
